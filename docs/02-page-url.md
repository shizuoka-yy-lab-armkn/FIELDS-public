# Page URL

- `/`: ホーム画面
    - 収録開始フォーム
    - 収録とその解析結果一覧

- `/recording/in-progress`: 収録中画面
    - 収録開始からの経過時間
    - 収録中断・完了フォーム
        - 中断ボタン
            - 収録を保存せずに終了
            - `/` へリダイレクト
        - 完了ボタン
            - 収録を保存して終了
            - `/` へリダイレクト

- `recordings/{id}?tab=(summary|video)`: 収録結果画面
    - 収録した動画と行動分節結果の統合表示
    - 解析結果に応じたスコアやその内訳の表示

- `/admin/users`: ユーザ一覧・管理画面
    - ユーザ一覧
    - ユーザの作成
    - ユーザ情報の修正
